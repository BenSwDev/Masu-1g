# סיכום פרוייקט מאסו (Masu) - פלטפורמת הזמנות טיפולים ביתיים

## אודות הפרוייקט
מאסו הוא פלטפורמה דיגיטלית מתקדמת להזמנת טיפולים ביתיים, המספקת פתרון מקיף לחיבור בין לקוחות למטפלים מקצועיים. המערכת בנויה כפלטפורמה רב-משתתפת עם ממשק ניהול מתקדם ותמיכה במספר שפות.

## טכנולוגיות ואדריכלות

### Frontend
- **Next.js 15.2.4** - Framework React מתקדם עם App Router
- **TypeScript** - פיתוח מוטיפוס לבטיחות קוד מקסימלית
- **Tailwind CSS** - עיצוב responsive ומודרני
- **Shadcn/ui** - ספריית קומפוננטים מתקדמת
- **React Hook Form + Zod** - ניהול טפסים ו-validation מחמיר
- **React Query (TanStack Query)** - ניהול state ו-caching

### Backend & Database
- **MongoDB** - מסד נתונים NoSQL עם Mongoose ODM
- **NextAuth.js** - מערכת אימות מתקדמת
- **API Routes** - RESTful APIs עם Next.js
- **Server Actions** - פעולות שרת מאובטחות

### Additional Technologies
- **i18n** - תמיכה רב-לשונית (עברית, אנגלית, רוסית)
- **Date-fns-tz** - ניהול זמנים ואזורי זמן
- **Crypto** - הצפנה ובטיחות נתונים
- **Vercel Deployment** - פריסה ו-CI/CD

## ארכיטקטורה ויציבות

### ביצועים ואופטימיזציה
